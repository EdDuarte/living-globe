        More Info

        Press the data types on the left panel to map another demographic indicator to the corresponding visual structure.

        Drag the slider handles on the left panel to limit the presented data according to new minimum and maximum values.

        Data mapped to the bar color uses colors from the gradient above, where blue represents lower values and yellow represents higher ones.

        Data mapped to the country color uses colors from the gradient above, where red represents lower values and green represents higher ones.


        Drag the slider below to change the year of collected data.

        Select a country on the globe, or write its name in the search field, to show detailed data for that country.
